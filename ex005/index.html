Semânitca: Negrito e Itálico

Principais formatações:

Negrito (ou destaque)

Há duas maneiras de negritar uma frase:

→ Usando a tag B
→ Ou deixando a frase em destaque usando a tag < strong >

Itálico

No itálico também detemos de duas formas de realizar esta marcação:

→ Utilizando a tag I (marcando a frase em itálico)
→ Ou podemos ulitizar a tag em (marcando a frase com destaque em itálico)

Texto Marcado

Há uma forma de marcar o texto que é ultilizando a tag mark

Textos pequenos

Para escrever um texto pequeno, colocamos a tag < small >

Textos deletados e inseridos

→ Para indicar que um texto foi deletado, colocamos a tag <del>

→ Para indicarque um texto foi inserido, colocamos a tag <ins>

Texto sobrescrito e subscrito

→ Para sobrescrevere um texto, colocamos a tag <sup> EX: 402=160

→Para subscrever um texto, colocamos a tag <sub> EX:H2O
